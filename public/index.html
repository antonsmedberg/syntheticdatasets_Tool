How to Use

To generate synthetic datasets for machine learning tasks, follow these steps:

  1. Specify the Number of Samples and Features:
    • Enter the desired number of samples (data points) and features (attributes) in the respective input fields.
    • Both values must be positive integers.
    • The number of samples should represent the size of the dataset (1 to 10,000).
    • The number of features indicates the dimensionality of each data point (1 to 1,000).
  2. Choose the Machine Learning Task:
    • Select the type of machine learning task from the dropdown menu:
    • Classification: Predict categories or classes.
    • Regression: Predict continuous values.
    • Clustering: Group similar data points.
  3. Generate and Save the Dataset:
    • Click the "Generate & Save Dataset" button to create the synthetic dataset based on your specifications.
    • The dataset will be saved as a CSV file named "generated_dataset.csv".
    • You can use this dataset for training machine learning models or experiments.
